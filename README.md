# Personal Portfolio | Rishik Raj P

A minimalist, high-performance personal portfolio built with modern CSS scroll-driven animations and a glassmorphism aesthetic. This site showcases my journey as a Computer Science Engineering student at Amrita Vishwa Vidyapeetham.

##  Live Website
[View Portfolio Here](https://rishik-0.github.io/Porfoliio-UID/docs/) 


## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 
- **Typography:** Anton, Italiana, Space Mono (via Google Fonts)
- **Design:** Dark Mode, Responsive UI

## ✨ Key Features
- **Scroll-Driven Animations:** Utilizes the CSS `animation-timeline: view()` API for high-performance fade and scale effects without JavaScript.
- **Infinite Skills Marquee:** A seamless, CSS-only horizontal scroll showcasing my tech stack (Python, React, Kotlin, C++, etc.).
- **Responsive Layout:** Fluid typography using `clamp()` and a flex-based grid to ensure compatibility from mobile to ultra-wide displays.
- **Project Showcase:** Interactive cards highlighting my work on MeloFi, Unix Shell, and AI bots.
